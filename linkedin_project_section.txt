DETAILED VERSION FOR PROJECT DESCRIPTION FIELD:

Developed a comprehensive blog platform and content management system as an individual university project for the module IN2120 – WEB PROGRAMMING. This full-stack web application demonstrates proficiency in modern web development practices and security implementation.

The platform features a three-tier user system (Admin, Author, User) with role-based access control. Admins can manage posts, users, and categories through an intuitive dashboard. Authors can create, edit, and publish posts with a rich text editor, while users can interact through likes, comments, and sharing functionality.

Technical implementation includes:
- Secure authentication system with bcrypt password hashing
- SQL injection prevention using prepared statements
- XSS protection through input sanitization
- RESTful API endpoints for AJAX operations
- MVC-like architecture for maintainable code structure
- Responsive, mobile-first design using Tailwind CSS
- Optimized database queries with proper indexing
- Environment-based configuration management

The project showcases ability to build production-ready web applications with emphasis on security, performance, and user experience.

Technologies: PHP, MySQL, JavaScript, Tailwind CSS, HTML5
